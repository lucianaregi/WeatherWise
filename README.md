# WeatherWise ???

## Descrição
WeatherWise é uma aplicação web que fornece dados meteorológicos detalhados e precisos, incluindo clima atual, previsão dos próximos dias e informações climáticas históricas. Utilizando tecnologias modernas como **ASP.NET Core**, **Angular** e integração com **APIs da OpenWeather**, o projeto visa facilitar o acesso a informações meteorológicas confiáveis para diferentes cidades e regiões.

## Objetivo
O objetivo principal do WeatherWise é criar uma plataforma intuitiva e visualmente atraente que permita aos usuários consultar condições climáticas em tempo real, acessar previsões detalhadas e visualizar dados meteorológicos históricos e em mapas interativos. A aplicação busca fornecer informações meteorológicas precisas de forma acessível, utilizando boas práticas de desenvolvimento e automação para garantir qualidade e desempenho.

## Tecnologias Utilizadas
- **Frontend**: Angular, Bootstrap
- **Backend**: ASP.NET Core, REST API
- **APIs Externas**: OpenWeather API
- **Infraestrutura**: Docker, Docker Compose
- **CI/CD**: GitHub Actions
- **Versionamento**: GitFlow no GitHub
